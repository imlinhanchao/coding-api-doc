开始

开始

要调用 Coding API，需要有访问 Token。根据不同的需求，需要不同的访问 Token。

只是操作自己的项目，可以使用个人令牌 Token。而如果是要做通用应用，就需要创建应用了。

另外，所有 API 均是在你的团队域名下，这意味着你无法创建跨团队的应用。
