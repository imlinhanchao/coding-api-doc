个人令牌

个人令牌

个人访问令牌，可以用于访问个人可以访问得到的项目内容，个人信息和消息通知。创建就可以直接得到一个访问 Token。然后访问 API 时，只需要在 Header 中加入 Authorization 的头信息: token YourToken 即可。

访问这里可以得到详细的创建个人令牌的方式。
